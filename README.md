# EUGVRP Security Bot 🛡️

Advanced security bot for Discord Roleplay servers. Built with `discord.js v14`.

## 🚀 Features
- **Anti-Raid:** Detects mass joins and acts automatically.
- **Anti-Bot:** Kicks unauthorized bots.
- **Shadowban System:** Special command for Support Team & Owners.
- **Lockdown:** Instantly lock the server in case of emergency.
- **Automated Logs:** All actions are sent to a dedicated channel.

## 🛠️ Deployment (Railway)
1. Create a new project on Railway.
2. Connect your GitHub repository.
3. Add the following **Variables**:
   - `DISCORD_TOKEN`
   - `CLIENT_ID`
4. Deploy!
